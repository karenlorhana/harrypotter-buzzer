# harrypotter-buzzer
 
<h2>Música tema de Harry Potter em um buzzer com o Arduino Uno</h2>
 
 
 <h3> Materiais utilizados</h3>
 
<ul>
<li> 1 Arduino Uno</li>
<li> 1 buzzer 5V</li>
<li> 1 protoboard</li>
<li> 2 LED's </li
<li> 3 resistores 220Ω</li
<li> Jumpers</li
</ul>
